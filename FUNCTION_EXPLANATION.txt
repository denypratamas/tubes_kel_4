FUNCTION EXPLANATION - CRYPTO MINING SIMULATION
================================================

File ini menjelaskan function yang ada di crypto_mining.go.
Penjelasan ini sudah disesuaikan dengan kondisi kode terbaru:
- Input hanya memakai fmt.
- Data memakai array statis dengan const NMAX = 100.
- Tidak memakai slice, append, make, bufio, os, atau strconv.
- Konversi angka dilakukan manual dengan stringKeInt() dan stringKeFloat().


DATA GLOBAL
-----------
- NMAX: kapasitas maksimal data aset dan sesi mining.
- masterData [NMAX]CryptoAsset: array utama untuk menyimpan aset crypto.
- miningSessions [NMAX]MiningSession: array untuk menyimpan riwayat simulasi mining.
- jumlahAset int: jumlah aset aktif yang terisi di masterData.
- jumlahMiningSessions int: jumlah sesi aktif yang terisi di miningSessions.
- computingPower float64: nilai computational power terakhir yang dimasukkan user.


1. initData()
-------------
Tujuan:
Mengisi data awal aset crypto ke dalam array global masterData.

Cara kerja:
- Mengisi masterData index 0 sampai 3 dengan Bitcoin, Zcash, Ethereum, dan Litecoin.
- Setiap aset memiliki nama, difficulty, estimasi reward, algoritma, konsumsi energi, dan status Mined.
- Mengatur jumlahAset menjadi 4.

Input:
Tidak ada parameter.

Output:
Tidak mengembalikan nilai.

Efek:
Mengubah isi masterData dan jumlahAset.


2. main()
---------
Tujuan:
Menjadi titik awal program saat dijalankan.

Cara kerja:
- Memanggil initData().
- Menampilkan menu utama secara berulang.
- Membaca pilihan user dengan bacaInt().
- Mengarahkan user ke menu sesuai pilihan.
- Program berhenti jika user memilih 0.

Efek:
Menjalankan seluruh alur aplikasi CLI.


3. kelolaAset()
---------------
Tujuan:
Menampilkan menu CRUD untuk mengelola aset crypto.

Cara kerja:
- Menampilkan daftar aset.
- Menampilkan pilihan tambah, edit, hapus, atau kembali.
- Memanggil tambahAset(), editAset(), atau hapusAset() sesuai pilihan user.

Efek:
Dapat mengubah isi masterData dan jumlahAset.


4. tampilkanSemuaAset()
-----------------------
Tujuan:
Menampilkan semua aset crypto yang aktif.

Cara kerja:
- Jika jumlahAset bernilai 0, tampilkan pesan bahwa belum ada aset.
- Jika ada data, loop dari index 0 sampai jumlahAset - 1.
- Untuk setiap aset, panggil tampilkanRingkasanAset().

Efek:
Menampilkan data aset ke layar.


5. tampilkanRingkasanAset(index int, aset CryptoAsset)
------------------------------------------------------
Tujuan:
Menampilkan satu baris ringkasan data aset.

Input:
- index int: posisi aset yang sedang ditampilkan.
- aset CryptoAsset: data aset yang ingin ditampilkan.

Cara kerja:
- Mengecek field Mined.
- Menampilkan nomor, nama, difficulty, reward, algoritma, energi, dan status mining.

Efek:
Menampilkan satu aset ke layar.


6. tambahAset()
---------------
Tujuan:
Menambahkan aset crypto baru ke masterData.

Cara kerja:
- Mengecek apakah jumlahAset sudah mencapai NMAX.
- Meminta input nama aset, difficulty, reward, algoritma, dan energi.
- Input string dibaca dengan bacaStringWajib().
- Input angka dibaca dengan bacaFloatPositif().
- Menyimpan data baru ke masterData[jumlahAset].
- Menambah jumlahAset sebanyak 1.

Efek:
Menambah satu data aset jika array belum penuh.


7. editAset()
-------------
Tujuan:
Mengubah data aset yang sudah ada.

Cara kerja:
- Mengecek apakah jumlahAset bernilai 0.
- Meminta nomor aset yang ingin diedit.
- Mengubah nomor menjadi index dengan rumus nomor - 1.
- Memvalidasi index dengan validasiIndex().
- Meminta data baru dari user.
- Mengganti data pada masterData[index].

Efek:
Mengubah salah satu data di masterData.


8. hapusAset()
--------------
Tujuan:
Menghapus aset dari masterData.

Cara kerja:
- Mengecek apakah jumlahAset bernilai 0.
- Meminta nomor aset yang ingin dihapus.
- Mengubah nomor menjadi index.
- Memvalidasi index.
- Menggeser semua data setelah index satu posisi ke kiri.
- Mengosongkan elemen terakhir yang sebelumnya aktif.
- Mengurangi jumlahAset sebanyak 1.

Efek:
Menghapus satu data aset dari array aktif.


9. validasiIndex(index int) bool
--------------------------------
Tujuan:
Memastikan index berada dalam batas data aktif masterData.

Cara kerja:
- Index valid jika index >= 0 dan index < jumlahAset.
- Jika tidak valid, tampilkan pesan error.

Output:
- bool: true jika valid, false jika tidak valid.


10. simulasiMining()
--------------------
Tujuan:
Menjalankan simulasi mining untuk satu aset pilihan user.

Cara kerja:
- Mengecek apakah jumlahAset bernilai 0.
- Mengecek apakah jumlahMiningSessions sudah mencapai NMAX.
- Menampilkan katalog aset.
- Meminta user memilih aset berdasarkan nomor atau nama.
- Meminta computational power.
- Menghitung multiplier algoritma, durasi, reward, energi, dan delay.
- Menjalankan time.Sleep(delay).
- Menyimpan hasil ke miningSessions[jumlahMiningSessions].
- Menambah jumlahMiningSessions sebanyak 1.
- Mengubah status aset menjadi pernah ditambang.
- Menampilkan hasil simulasi.

Efek:
Menambah riwayat mining dan mengubah field Mined pada aset.


11. pilihAsetUntukMining() int
------------------------------
Tujuan:
Membaca pilihan aset dari user untuk simulasi mining.

Cara kerja:
- Meminta user memasukkan nomor atau nama aset.
- Mencoba mengubah input menjadi angka dengan stringKeInt().
- Jika input angka, angka diubah menjadi index.
- Jika input bukan angka, program mencari aset berdasarkan nama dengan sequentialSearch().

Output:
- int: index aset yang dipilih.
- Mengembalikan -1 jika aset tidak ditemukan.


12. hitungAlgoMultiplier(algorithm string) float64
--------------------------------------------------
Tujuan:
Memberikan nilai multiplier berdasarkan jenis algoritma mining.

Cara kerja:
- Mengubah nama algoritma menjadi huruf kecil dengan ubahKeHurufKecil().
- Mencocokkan algoritma dengan sha-256, ethash, scrypt, atau equihash.
- Mengembalikan multiplier default 1.0 jika algoritma tidak dikenali.

Output:
- float64: multiplier algoritma.


13. hitungDelayMining(aset CryptoAsset, power float64, algoMultiplier float64) time.Duration
-------------------------------------------------------------------------------------------
Tujuan:
Menghitung delay buatan untuk simulasi mining.

Cara kerja:
- rawDelay = (aset.Difficulty * algoMultiplier) / power.
- rawDelay dikali 1.5.
- Delay dibatasi minimal 1 detik dan maksimal 7 detik.
- Hasil dikonversi menjadi time.Duration.

Output:
- time.Duration: durasi delay untuk time.Sleep().


14. menuCariAset()
------------------
Tujuan:
Menjalankan fitur pencarian aset berdasarkan nama.

Cara kerja:
- Mengecek apakah jumlahAset bernilai 0.
- Meminta nama aset dari user.
- Menyalin masterData dengan salinData().
- Mengurutkan salinan data berdasarkan nama dengan insertionSortByName().
- Mencari aset dengan binarySearch().
- Menampilkan hasil jika ditemukan.

Efek:
Menampilkan hasil pencarian. Data asli masterData tidak berubah.


15. sequentialSearch(keyword string) int
---------------------------------------
Tujuan:
Mencari aset berdasarkan nama dengan cara mengecek satu per satu.

Cara kerja:
- Loop dari index 0 sampai jumlahAset - 1.
- Membandingkan nama aset dengan keyword memakai samaTanpaHurufBesarKecil().
- Mengembalikan index jika ditemukan.
- Mengembalikan -1 jika tidak ditemukan.

Catatan:
Sequential search memiliki kompleksitas O(n).


16. binarySearch(data [NMAX]CryptoAsset, jumlah int, keyword string) int
------------------------------------------------------------------------
Tujuan:
Mencari aset berdasarkan nama dengan Binary Search.

Input:
- data [NMAX]CryptoAsset: array aset yang sudah terurut berdasarkan nama.
- jumlah int: jumlah elemen aktif yang perlu diproses.
- keyword string: nama aset yang dicari.

Cara kerja:
- Menentukan batas kiri dan kanan berdasarkan jumlah.
- Mengambil index tengah.
- Membandingkan keyword dengan nama aset di tengah.
- Jika sama, index tengah dikembalikan.
- Jika keyword lebih besar, pencarian dilanjutkan ke kanan.
- Jika keyword lebih kecil, pencarian dilanjutkan ke kiri.
- Jika tidak ditemukan, mengembalikan -1.

Catatan:
Binary Search harus memakai data yang sudah terurut.
Kompleksitasnya O(log n).


17. menuUrutkanAset()
---------------------
Tujuan:
Menjalankan fitur pengurutan aset.

Cara kerja:
- Mengecek apakah jumlahAset bernilai 0.
- Menampilkan pilihan pengurutan berdasarkan difficulty atau reward.
- Difficulty memanggil selectionSortByDifficulty(&masterData, jumlahAset).
- Reward memanggil insertionSortByReward(&masterData, jumlahAset).
- Menampilkan data setelah diurutkan.

Efek:
Mengubah urutan data aktif di masterData.


18. selectionSortByDifficulty(data *[NMAX]CryptoAsset, jumlah int)
------------------------------------------------------------------
Tujuan:
Mengurutkan aset berdasarkan difficulty dari kecil ke besar.

Input:
- data *[NMAX]CryptoAsset: pointer ke array aset yang ingin diurutkan.
- jumlah int: jumlah elemen aktif.

Cara kerja:
- Mulai dari posisi pertama.
- Cari aset dengan difficulty terkecil di sisa data aktif.
- Tukar aset terkecil ke posisi saat ini.
- Ulangi sampai data terurut.

Catatan:
Selection Sort memiliki kompleksitas O(n^2).


19. insertionSortByReward(data *[NMAX]CryptoAsset, jumlah int)
--------------------------------------------------------------
Tujuan:
Mengurutkan aset berdasarkan estimated reward dari besar ke kecil.

Input:
- data *[NMAX]CryptoAsset: pointer ke array aset yang ingin diurutkan.
- jumlah int: jumlah elemen aktif.

Cara kerja:
- Menganggap bagian kiri array sudah terurut.
- Mengambil satu data sebagai key.
- Menggeser data dengan reward lebih kecil ke kanan.
- Menyisipkan key ke posisi yang benar.

Catatan:
Insertion Sort memiliki kompleksitas O(n^2).


20. insertionSortByName(data *[NMAX]CryptoAsset, jumlah int)
------------------------------------------------------------
Tujuan:
Mengurutkan aset berdasarkan nama dari A ke Z.

Cara kerja:
- Menggunakan konsep Insertion Sort.
- Membandingkan nama aset dalam bentuk huruf kecil.
- Menggeser data yang namanya lebih besar secara alfabetis.
- Menyisipkan data ke posisi yang tepat.

Catatan:
Function ini dipakai sebelum binarySearch().


21. salinData(data [NMAX]CryptoAsset, jumlah int) [NMAX]CryptoAsset
-------------------------------------------------------------------
Tujuan:
Membuat salinan array aset.

Cara kerja:
- Membuat variabel lokal hasil bertipe [NMAX]CryptoAsset.
- Loop dari index 0 sampai jumlah - 1.
- Menyalin setiap elemen aktif dari data ke hasil.
- Mengembalikan array hasil salinan.

Efek:
Tidak mengubah data asli.


22. bacaString(prompt string) string
------------------------------------
Tujuan:
Membaca input teks dari user hanya dengan fmt.

Cara kerja:
- Menampilkan prompt ke layar.
- Membaca input per karakter memakai fmt.Scanf("%c", &karakter).
- Berhenti saat menemukan newline.
- Mengabaikan carriage return.
- Merapikan input dengan trimSpasi().

Catatan:
Karena membaca per karakter, input seperti "Bitcoin Cash" tetap bisa dibaca sebagai satu string.


23. bacaStringWajib(prompt string) string
-----------------------------------------
Tujuan:
Membaca input teks yang tidak boleh kosong.

Cara kerja:
- Memanggil bacaString().
- Jika input kosong, tampilkan error.
- Ulangi sampai user memasukkan teks yang valid.


24. bacaInt(prompt string) int
------------------------------
Tujuan:
Membaca input angka bulat dari user.

Cara kerja:
- Membaca input dengan bacaStringWajib().
- Mengubah input menjadi int memakai stringKeInt().
- Jika berhasil, nilai dikembalikan.
- Jika gagal, tampilkan error dan ulangi input.


25. bacaFloatPositif(prompt string) float64
-------------------------------------------
Tujuan:
Membaca input angka desimal positif dari user.

Cara kerja:
- Membaca input dengan bacaStringWajib().
- Mengubah input menjadi float64 memakai stringKeFloat().
- Memastikan angka lebih dari 0.
- Jika input valid, nilai dikembalikan.
- Jika input tidak valid, tampilkan error dan ulangi input.


26. stringKeInt(teks string) (int, bool)
----------------------------------------
Tujuan:
Mengubah string menjadi int tanpa strconv.

Cara kerja:
- Membersihkan spasi awal dan akhir.
- Mengecek tanda minus atau plus di awal.
- Memastikan semua karakter berikutnya adalah angka 0 sampai 9.
- Menghitung nilai integer secara manual.
- Mengembalikan nilai dan status keberhasilan.

Output:
- int: hasil konversi.
- bool: true jika konversi berhasil, false jika gagal.


27. stringKeFloat(teks string) (float64, bool)
----------------------------------------------
Tujuan:
Mengubah string menjadi float64 tanpa strconv.

Cara kerja:
- Membersihkan spasi awal dan akhir.
- Mengecek tanda minus atau plus di awal.
- Membaca angka sebelum dan sesudah titik desimal.
- Menolak input yang punya lebih dari satu titik desimal.
- Mengembalikan nilai dan status keberhasilan.

Output:
- float64: hasil konversi.
- bool: true jika konversi berhasil, false jika gagal.


28. trimSpasi(teks string) string
---------------------------------
Tujuan:
Menghapus spasi di awal dan akhir teks.

Cara kerja:
- Mencari index awal yang bukan karakter spasi.
- Mencari index akhir yang bukan karakter spasi.
- Mengembalikan potongan teks dari awal sampai akhir.
- Jika teks hanya berisi spasi, mengembalikan string kosong.


29. teksKosong(teks string) bool
--------------------------------
Tujuan:
Mengecek apakah teks kosong atau hanya berisi spasi.

Cara kerja:
- Memanggil trimSpasi(teks).
- Jika hasilnya string kosong, berarti teks kosong.


30. karakterSpasi(karakter byte) bool
-------------------------------------
Tujuan:
Mengecek apakah satu karakter termasuk spasi.

Cara kerja:
- Mengembalikan true jika karakter adalah spasi, tab, newline, atau carriage return.
- Jika bukan salah satu dari itu, mengembalikan false.


31. samaTanpaHurufBesarKecil(teks1 string, teks2 string) bool
-------------------------------------------------------------
Tujuan:
Membandingkan dua teks tanpa memperhatikan huruf besar dan kecil.

Cara kerja:
- Menghapus spasi awal dan akhir dari kedua teks.
- Mengubah kedua teks menjadi huruf kecil dengan ubahKeHurufKecil().
- Membandingkan kedua hasilnya.


32. ubahKeHurufKecil(teks string) string
----------------------------------------
Tujuan:
Mengubah huruf kapital A-Z menjadi huruf kecil.

Cara kerja:
- Membaca string dari index 0 sampai akhir.
- Jika karakter berada di antara A sampai Z, karakter ditambah 32.
- Hasil disusun kembali sebagai string baru.

Catatan:
Function ini memakai operasi ASCII sederhana dan tidak memakai slice rune.


33. tampilkanLaporan()
----------------------
Tujuan:
Menampilkan laporan hasil semua sesi mining.

Cara kerja:
- Mengecek apakah jumlahMiningSessions bernilai 0.
- Jika kosong, tampilkan pesan agar user menjalankan simulasi dulu.
- Loop dari index 0 sampai jumlahMiningSessions - 1.
- Menghitung total reward, total durasi, dan total energi.
- Mencari sesi dengan reward terbesar sebagai aset paling menguntungkan.
- Menghitung rata-rata durasi.
- Menampilkan semua hasil laporan ke layar.

Efek:
Menampilkan laporan mining ke layar.
